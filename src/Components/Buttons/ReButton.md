Primary Button

```js
<ReButton
  text="Click Me"
  type="primary"
  onClick={() => console.log('Clicked')}
/>
```

Secondary Button

```js
<ReButton
  text="Click Me"
  type="secondary"
  onClick={() => console.log('Clicked')}
/>
```

Add the loading modifier

```js
<ReButton text="Loading" type="primary" loading />
```

Primary Button

```js
<Button
  variant="primary"
  onClick={() => alert('Clicked!')}
>Click Me</Button>
```

Secondary Button

```js
<Button
  variant="secondary"
  onClick={() => alert('Clicked!')}
>Click Me</Button>
```

Add the loading modifier

```js
<Button variant="primary" loading>Loading</Button>
```

Add the disabled modifier

```js
<Button variant="primary" disabled>Disabled</Button>
```

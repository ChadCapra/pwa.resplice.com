Primary Button

```js
<ReButton
  type="primary"
  onClick={() => console.log('Clicked')}
>Click Me</ReButton>
```

Secondary Button

```js
<ReButton
  type="secondary"
  onClick={() => console.log('Clicked')}
>Click Me</ReButton>
```

Add the loading modifier

```js
<ReButton type="primary" loading>Loading</ReButton>
```

Add the disabled modifier

```js
<ReButton type="primary" disabled>Disabled</ReButton>
```
